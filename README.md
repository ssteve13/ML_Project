# 🎵 Moodify — Emotion-Based Music Recommender

> AI-powered music tuned to your mood. Built with Streamlit, scikit-learn, and YouTube.

---

## ✨ Features

| Feature | Description |
|---|---|
| **Time-aware presets** | Automatically picks a mood based on the current time of day (e.g. ☀️ Morning, 🌙 Late Night) |
| **11 Mood modes** | Happy, Sad, Chill, Energetic, Romantic, Focus, Party, Angry, Late Night, Morning, Nostalgic |
| **Fuzzy seed search** | Type song names (even with typos) and Moodify will find them using `rapidfuzz` |
| **AI emotion detection** | A Random Forest classifier maps your audio feature vector to Happy / Sad / Calm / Energetic |
| **Fine-tune sliders** | Adjust Energy, Danceability, Positivity, Acousticness, Tempo, and number of results |
| **In-page YouTube player** | Embeds YouTube directly — no redirect needed |
| **Favourites panel** | Heart any song to save it; replay from the favourites panel anytime |

## 🚀 Getting Started

### 1. Install dependencies

```bash
pip install -r requirements.txt
```

### 2. Ensure models and data are present

Make sure the following files exist before running the app:

- `models/emotion_model.pkl`
- `models/scaler.pkl`
- `data/songs.csv`

> These are generated by the training notebook in `notebook/`. Run it first if they are missing.

### 3. Run the app

From the **project root** (`music/`):

```bash
streamlit run app/app.py
```

The app will open in your browser at `http://localhost:8501`.

## 🛠️ How It Works

1. **Mood selection** — choose from 11 mood presets or let time-of-day auto-select one.
2. **Seed songs** *(optional)* — type song names to blend their audio profile into your search (60/40 weighted blend).
3. **Fine-tune** — adjust sliders for energy, tempo, danceability, etc.
4. **Discover** — the Random Forest model predicts your emotion class, then ranks songs from the dataset by how closely they match your chosen audio features.
5. **Play** — click ▶ to embed the YouTube video in-page, or heart it to add to Favourites.
